Technology Stack

Core Components

  • Flask - Python web framework handling all HTTP requests and UI rendering
  • ClamAV - Open-source antivirus engine providing signature-based detection
  • Paramiko - SSH library for remote command execution and file transfers
  • LIEF - Library for parsing and analyzing executable formats

Analysis Engines

  • YARA - Pattern matching for malware identification
  • Unicorn Engine - CPU emulation for dynamic analysis
  • Capstone/Keystone - Disassembly and assembly frameworks
  • NetworkX - Network topology visualization
