How This Works
Remote Installation Process

This tool automates ClamAV installation across multiple hosts using SSH (Linux) or WinRM (Windows).

Technical Details
  • Uses Paramiko for SSH connections
  • Detects OS automatically
  • Parallel installation on multiple hosts
  • Real-time progress streaming
Supported Systems
Linux
  • Ubuntu/Debian
  • CentOS/RHEL
  • Fedora
  • OpenSUSE
Windows
  • Windows 10/11
  • Windows Server
Requirements
  • SSH access (Linux)
  • Admin privileges (Windows)
  • Python 3.6+ on target hosts
  • Network connectivity
