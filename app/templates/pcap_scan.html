What is a PCAP File?

A Packet Capture (PCAP) file contains raw network traffic data recorded over a period of time. These files are generated by tools like Wireshark, Tcpdump, or intrusion detection systems.

In Cybersecurity:
  • Investigating security incidents
  • Detecting malware communications
  • Analyzing network performance
  • Applying threat detection rules
Supported Features:
  • Protocol identification (TCP, UDP, ICMP)
  • HTTP request/response analysis
  • DNS query inspection
  • Malicious pattern detection
File Requirements:
  • Formats: .pcap, .pcapng
  • Max size: 100MB
